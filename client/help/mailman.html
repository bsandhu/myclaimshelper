HOW-TO Submit Claim entries via e-mail

  1. Send/forward emails to put@mailman.logicubesolutions.com
  2. Add claim id: xyz to the subject line, where xyz is the Claim ID to which the entry belongs.
  3. Optionally, add #tags to the body of the message to add arbitrary tags/key words to the entry.
  4. Submit!

Example

To:put@mailman.logicubesolutions.com
Subject:FW: Bill, please investigate claim id: 123
Body:Bill, this looks like fun!
#car #nyc
